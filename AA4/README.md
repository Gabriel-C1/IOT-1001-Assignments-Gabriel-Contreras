This program lets the user manage up to 10 tasks using a menu-driven interface.  
Users can add, delete, complete, and view tasks.

## Added Features for Applied Activity 6 (Q2)
- **Export ToDo List:** Saves all tasks to `todo_export.txt`
- **Import ToDo List:** Loads tasks from `todo_export.txt` into the program
- Ensures tasks are loaded safely into the 10-slot array

A sample `todo_export.txt` file is included.

## How to Run
1. Open the project in Visual Studio.
2. Run the program.
3. Use the menu options to manage tasks.
4. Choose "Export" or "Import" to test file features.
